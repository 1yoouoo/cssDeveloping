1. 인라인 요소
    -> 기본 너비 값 : 컨텐츠의 너비 값 / 한줄에 여러 개 배치 / 크기 값을 가질 수 없음 / 상하 마진은 가질 수 없음

2. 블록 요소
    -> 기본 너비 값 :100% / 한 줄에 하나만 배치 / 너비 값, 높이 값 가질 수 있음 / 상하좌우 마진 가질 수 있음

3. 인라인 블록 요소
    -> 기본 너비 값 : 컨텐츠의 너비 값 / 한 줄에 여러 개 배치 / 크기 값을 가질 수 있음 / 상하좌우 마진 가질 수 있음


// 요소를 보이지 않게 하는 3가지 방법과 차이점 ..

1. display : none ;
    -> 존재 자체를 사라지게 함 
    
2. opacity : 0 ;
    -> 존재 자체는 유지한 채 투명도의 조절로 보이지 않게 함

3. visibility : hidden ;
    -> 자기 자리의 크기와 위치는 유지한 채 사라지게 함
